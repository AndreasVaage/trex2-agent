High level decision and control for autonomous agent

This project is taking its roots from the same initial code base as trex-autonomy project which was maintained by Willow Garage. Both projects were issued from initial design ideas developed at MBARI to bring goal directed control and deliberation on autonomous robots.

The reason to start this new project (initially hosted on Google code) was 2 folds:

  • trex-autonomy project became stale when Conor McGann left Willow Garage and is no longer supported
  • further this new version wanted to get closer to the formal description of T-REX given in our AAMAS publication (A Systematic Agent Framework for Situated Autonomous Systems. F. Py, K. Rajan & C. McGann 9th Int. Conf. on Autonomous Agents and Multiagent Systems (AAMAS) , May 2010, Toronto, Canada)

Finally while former versions of TREX were tied up to the europa-pso planner, this new version is independent of the planner that being used for the deliberation inside the reactors that are the basic building blocks of an agent. Even though, europa-pso is the only planner publicly supported here, it was moved as a plug-in and compiling the core part of t-rex do not require to install europa.

Requirements

T-REX code is written mostly in C++ and require a modern compiler with, optionally, support for c++11 (for example it uses new std::chrono classes but will try to uses its boost counterpart by default). while the code is trying to remain as system agnostic as possible it may rely on some POSIX constructs and has been only tested on POSIX compliant systems (Linux and MacOS X).

Build management is done using cmake

The core library only requires a recent version of the boost and the earlier version supported is 1.47.0.

The default build of T-REX will also compile the europa plugin which requires europa-pso version 2.6 (there's some limited support for 2.5 but it has not been tested for a long time and is not as flexible)

Code documentation is done using doxygen

Other plug-in provided can have further dependencies which we do not detail on this page.
